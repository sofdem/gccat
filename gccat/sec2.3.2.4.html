Graph terminology and notations

A digraph G=(V(G),E(G)) is a pair where V(G) is a finite set, called the set of vertices, and where E(G) is a set of ordered pairs of vertices, called the set of arcs. The arc, path, circuit and strongly connected component of a graph G correspond to oriented concepts, while the edge, chain, cycle and connected component are non-oriented concepts. However, as reported inÂ [Berge70] an undirected graph can be seen as a digraph where to each edge we associate the corresponding two arcs.
